College Enquiry Chatbot

Develop a web application. It provide information about college admission process, course available, location, faculty, campus, hostel enquiry, resourse available for study, placements etc.,

Implemented By using Python with Flask.

Use MySQL database to securely store user data and session logs.

For better user interface HTML, CSS, Boostrap is used.

User Guideline

  1. Getting Started: First go to the website home page.
  2. Registration: Once you're on the website, look for the "Register" button. Click on it and fill out the registration form with your details, like your name, email, and a password.
  3. Logging In: Now that you're registered, it's time to log in. then you will redirect to login page. Enter your name and password, and youre in!
  4. Chat Section: You're now logged in and ready to explore. Navigate to the chat section, where you can find the chatbot waiting to help you out. First greet the chatbot with hi or hello and then just type a choice number from the suggested topics , and the chatbot will provide all the information that you need. If you need a menu then type 0.
  5. Providing Feedback: If you have any feedback or suggestions for us, You can easily provide feedback by clicking on the feedback button and type your thoughts.
  6. Update profile : User ca also update their profile by changing the information of email, phone address ect.
  7. Logging Out: Once you're done chatting and exploring, it's time to log out. Look for the "Log Out" button,Click on it, and you'll be safely logged out of your account.
